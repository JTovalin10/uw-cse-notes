# Parallelism

**[[Parallelism]]** is the simultaneous execution of multiple tasks at the exact same physical time, typically requiring multiple CPU cores or processors.

## Key Concepts
- **Speedup**: The ratio of time taken on one processor ($T_1$) to the time taken on $P$ processors ($T_P$).
- **Span ($T_\infty$)**: The time taken by the longest path of dependencies in a parallel algorithm.
- **Work ($T_1$)**: The total amount of time taken by all tasks in a parallel algorithm if executed sequentially.
- **Amdahl's Law**: A formula used to find the maximum improvement possible by improving a particular part of a system.
