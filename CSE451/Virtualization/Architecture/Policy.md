# CSE451: Policy

**Policy** describes *what* you are trying to achieve — the goal or rule that guides system decisions.

Contrast with [[Mechanism]], which defines *how* that goal is carried out.

## Related
- [[Mechanism]] — the implementation that enforces policy
- [[Scheduling]] — scheduling algorithms are policies for CPU time allocation
