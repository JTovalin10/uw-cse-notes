**Hypertext Transport Protocol**
- a request/response protocol
	- client sends a requests to a web server
	- the server processes the request and sends a response
- a request asks a server to retrieve a resource
	- a resource is an object or document named by a Uniform Resource Identifier (URI)
- A response indicates whether or not the server succeeded
	- it provided content of the requested response if it succeeded
# methods
## most common
- GET - get data
- POST - upload data
  HEAD - sends the headers of hte named resources
