# CSE 311: Proposition

A **proposition** is a statement that is "well-formed" and is either true or false. Being well-formed means it is a complete declarative statement, and having a definite truth value is what distinguishes a proposition from other kinds of sentences.

- Example: $2 + 2 = 5$ is a proposition (it is well-formed and has a truth value — it is false).
- Example: "Who are you?" is not a proposition (it is a question, so it has no truth value).

Propositions are the atomic building blocks of [[CSE311/Part I - Mathematical Foundations/Logic/Boolean Algebra|propositional logic]]. When a statement instead depends on a variable and only becomes true or false once that variable is filled in, it is a [[CSE311/Part I - Mathematical Foundations/Predicates and Quantifiers/What is a Predicate|predicate]] rather than a proposition.
