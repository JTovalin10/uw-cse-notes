# CSE 311: Lists (Notation)

The `::` operator (read "cons") prepends an element to a list:

$$x \mathbin{::} L \quad \text{means} \quad \text{cons}(x, L)$$

**Examples:**

- `nil` = `[]` (the empty list)
- `cons(3, nil)` = `3::nil` = `[3]`
