# CSE 311: List of Integers

A **List** is defined inductively:

**Basis:** $\text{nil} \in \text{List}$

**Recursive Step:** if $L \in \text{List}$ and $a \in \mathbb{Z}$, then $a \mathbin{::} L \in \text{List}$
