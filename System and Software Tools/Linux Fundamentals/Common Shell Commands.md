# CSE391: Common Shell Commands

These are essential utility commands used throughout the course for file manipulation and data processing.

| Command | Description |
|---------|-------------|
| **[[cat\|cat]]** | Concatenate and print files to stdout. |
| **[[echo\|echo]]** | Print text or variables to the terminal. |
| **[[grep\|grep]]** | Search text using regular expressions. |
| **[[Sed Commands\|sed]]** | Stream editor for filtering and transforming text. |
| **[[sort\|sort]]** | Sort lines of text files. |
| **[[uniq\|uniq]]** | Report or omit repeated lines (requires sorted input). |
| **[[wc\|wc]]** | Print newline, word, and byte counts for each file. |
| **[[head\|head]]** | Output the first part of files. |
| **[[tail\|tail]]** | Output the last part of files. |
| **[[Find and Cut#find\|find]]** | Search for files in a directory hierarchy. |
| **[[Find and Cut#cut\|cut]]** | Remove sections from each line of files. |
