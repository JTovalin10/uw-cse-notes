# Basic Data Types in Math

In math, basic data types are "sets". Sets are collections of objects called elements.
- We write $x \in S$ to say that "$x$" is an element of set "$S$", and $x \notin S$ to say that it is not.

Examples of basic sets:
- $\mathbb{Z}$ : the set of integers
- $\mathbb{N}$ : the set of non-negative integers (natural numbers)
- $\mathbb{R}$ : the set of real numbers
- $\mathbb{B}$ : the set of booleans ($T$ or $F$). Note: this is a non-standard name often used in this course.

We often write $x : \mathbb{Z}$ instead of $x \in \mathbb{Z}$ when defining types.
