# CSE369: Introduction to Digital Design

Introductory course to the world of digital logic and hardware design. We learn how to build the primitives that make up a computer, starting from logic gates and moving up to basic CPU components.

## Prerequisites
- Building machines that manipulate binary data.
- Knowledge of binary representation (from [[CSE351/Number Representation|CSE351]]).

## Hardware
- **[[CSE369/Memory and FPGAs|FPGA]]**: Field-Programmable Gate Array (Altera Cyclone V).
- **[[CSE369/Verilog Fundamentals|Verilog]]**: Hardware Description Language used for synthesis.
