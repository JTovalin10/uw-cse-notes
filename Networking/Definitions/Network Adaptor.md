# CSE461: Network Adaptor

A **network adaptor** is a piece of hardware that connects a node to a [[Link]]. Contains a signalling component that encodes bits into signals at the sending node and decodes signals back into bits at the receiving node.

See also: [[Network Interface Card (NIC)]] — the modern, more specific term for a network adaptor.
