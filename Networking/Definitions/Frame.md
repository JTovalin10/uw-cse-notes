# CSE461: Frame

A **frame** is the unit of data at the data link (Link) layer. It is a collection of bits grouped into a message that can be delivered to an adjacent node. Includes a header (addressing and control info) and a payload, often with error detection (e.g., [[Cyclic Redundancy Check (CRC)|CRC]]) appended.
