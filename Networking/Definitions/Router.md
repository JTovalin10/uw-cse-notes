# CSE461: Router

A **router** (also called an **intermediate system** or **gateway**) is a network device that relays packets between links by reading the destination address and forwarding each packet toward its destination. Routers connect multiple networks together.

Examples: access points, cable/DSL modems, enterprise core routers.
