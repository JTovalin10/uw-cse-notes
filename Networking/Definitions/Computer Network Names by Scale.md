# CSE461: Computer Network Names by Scale

Networks are commonly named by their geographic or organizational scope:

| Name | Scope | Example |
|------|-------|---------|
| **PAN** (Personal Area Network) | ~1 m | Bluetooth devices |
| **LAN** (Local Area Network) | Room / Building | Ethernet, Wi-Fi |
| **MAN** (Metropolitan Area Network) | City | Cable TV networks |
| **WAN** (Wide Area Network) | Country / Continent | Internet backbone |
| **Internet** | Global | The public Internet |
