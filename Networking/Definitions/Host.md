# CSE461: Host

A **host** (also called **end system**, **edge device**, **source**, or **sink**) is a computer at the edge of the network that runs user applications. Hosts generate and consume traffic; they do not relay packets for others.

Examples: laptop, mobile phone, desktop computer, server.
