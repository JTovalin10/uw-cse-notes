# CSE461: Signalling

**Signalling** is the process where a host sends a control message into the network to dynamically establish a [[Virtual Circuit (VC)]]. The message propagates switch-by-switch, creating connection state along the path.

Results in a [[Switched Virtual Circuit]]. Contrast with a [[Permanent Virtual Circuit]], which is manually configured by an administrator.
