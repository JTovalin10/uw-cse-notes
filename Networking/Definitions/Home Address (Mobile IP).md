# Home Address (Mobile IP)

In [[Mobile IP]], the **home address** is the mobile node's permanent [[IP Address]] — the prefix of its home network. Correspondents always send to this address; the [[Home Agent (Mobile IP)|home agent]] intercepts and tunnels to the current [[Care-of Address (Mobile IP)|care-of address]].
