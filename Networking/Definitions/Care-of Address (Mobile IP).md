# Care-of Address (Mobile IP)

In [[Mobile IP]], the **care-of address** is a temporary [[IP Address]] on the **foreign network** where the mobile node is currently attached. It changes when the mobile moves.

## Key points

- Temporary; reflects current location
- The [[Home Agent (Mobile IP)|home agent]] tunnels packets to this address
- **Co-located care-of**: mobile gets its own IP on the foreign network (e.g., via DHCP) and acts as its own "foreign agent"
