# Home Agent (Mobile IP)

In [[Mobile IP#Agents|Mobile IP]], the **home agent** is a router on the mobile node's **home network** that intercepts packets destined for the mobile's [[Home Address (Mobile IP)|home address]] and tunnels them to the mobile's current [[Care-of Address (Mobile IP)|care-of address]].

## Key points

- Located on the home network
- Uses proxy ARP to intercept packets for the mobile node
- Tunnels packets to the [[Foreign Agent (Mobile IP)|foreign agent]] or mobile's care-of address
