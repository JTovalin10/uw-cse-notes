# Network Layer Overview

Fixes scalability limits of the link layer. Provides:
- **Hierarchy** — scale beyond switches
- **Heterogeneity** — IP for internetworking across different technologies
- **Bandwidth control** — lowest-cost routing, QoS

## Subdirectories

- [[Internet (IP)]] — IP protocol, addressing, packets, protocols, routers
- [[Network Services]] — Datagrams vs virtual circuits, store-and-forward
- [[ATM]] — Asynchronous Transfer Mode
- [[Implementation]] — Software/hardware switches, SDN, NOS
- [[Source Routing]] — Source routing options
