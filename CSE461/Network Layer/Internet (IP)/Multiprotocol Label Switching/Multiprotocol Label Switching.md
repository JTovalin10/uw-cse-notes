# Multiprotocol Label Switching (MPLS)

Combines **label-swapping** (like virtual circuits) with **IP routing**. Forward by short fixed-length label instead of longest-match IP lookup.

**Uses:** IP on non-IP devices (ATM, optical), explicit routing, VPNs.
