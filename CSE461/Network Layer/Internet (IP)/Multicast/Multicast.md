# Multicast

IP-level multicast: send **one** packet to a group; routers duplicate at branch points. Scalable for one-to-many (IPTV, radio) and many-to-many (videoconference).

**Unicast** = one-to-one. **Multicast** = one-to-many or many-to-many.
