Supporting protocols that handle address resolution, configuration, and error reporting for IP networks.

## Topics

- [[IP Address Resolution Protocol (ARP)]] — resolves **IPv4** L3 addresses to L2 (MAC) addresses using broadcast queries and a local cache
- [[Neighbor Discovery Protocol (NDP)]] — resolves **IPv6** L3 addresses to L2 (MAC) addresses; IPv6's replacement for ARP, built on ICMPv6 using multicast
- [[Dynamic Host Configuration Protocol (DHCP)]] — automatically assigns IP addresses and configuration to hosts on demand
- [[Internet Control Message Protocol (ICMP)]] — reports errors back to the source when a router cannot deliver a datagram

See also: [[IP Datagram Forwarding]], [[Internetwork (IP)]]
