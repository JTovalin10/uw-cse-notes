# Full-duplex
- bidirectional at the same exact time
- Examples: modern Ethernet, telephone calls
- Both parties can send and receive simultaneously

# Half-duplex
- bidirectional but not at the same exact time
- Examples: walkie-talkies, WiFi (802.11)
- Must take turns transmitting

# Simplex
- unidirectional only
- Examples: TV broadcast, keyboard to computer
- One sender, one or more receivers

# Comparison Table
| Mode | Direction | Simultaneous | Example |
|------|-----------|--------------|---------|
| Simplex | One-way | N/A | TV broadcast |
| Half-duplex | Two-way | No | Walkie-talkie |
| Full-duplex | Two-way | Yes | Phone call |
