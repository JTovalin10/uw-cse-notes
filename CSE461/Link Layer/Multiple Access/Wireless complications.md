## Nodes
Nodes can't hear while sending; as with wires, detecting collisions (and aborting) lowers their costs. This means it has more wasted time with wireless.

Both communicating nodes need to send the same power in order to hear each other. Think: if you are yelling at someone you probably can't hear someone whispering to you; instead, if they are also yelling then you can probably hear them.
