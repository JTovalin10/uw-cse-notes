# CSE461: Baud Rate

The **baud rate** is the rate at which the signal changes on a link (measured in symbols per second). Not the same as bit rate — one signal change (symbol) can encode multiple bits depending on the modulation scheme.

$$\text{Bit Rate} = \text{Baud Rate} \times \log_2(\text{Number of Signal Levels})$$
