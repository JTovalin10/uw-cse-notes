# CSE461: Loose Source Route

A **loose source route** is a type of [[Source Routing]] where only a set of waypoints (intermediate nodes) is specified — how to get between them is left to the network's normal routing.

- Reduces the topology knowledge the source needs
- Helpful in large networks where acquiring complete path information for a [[Strict Source Route]] is difficult
