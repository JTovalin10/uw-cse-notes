## User Datagram Protocol (UDP)
A transport-layer protocol that provides **unreliable datagram delivery** — no guarantees on ordering, delivery, or duplicate detection. Faster than [[CSE461/Definitions/Transmission Control Protocol (TCP)|TCP]] because it skips connection setup and reliability overhead.

One of the two main transport protocols in the Internet architecture, alongside [[CSE461/Definitions/Transmission Control Protocol (TCP)|TCP]].
