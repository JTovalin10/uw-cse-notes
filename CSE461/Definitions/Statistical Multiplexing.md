# CSE461: Statistical Multiplexing

**Statistical Multiplexing** is a [[Multiplexing]] approach that shares the physical link on demand rather than in predetermined slots. Data is transmitted from each flow only when that flow has data to send. If only one flow has data, it gets the full link without waiting. Addresses the wasted-capacity shortcomings of [[Synchronous Time-Division Multiplexing]] and [[Frequency Division Multiplexing (FDM)]].
