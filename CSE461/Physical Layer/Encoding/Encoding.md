## Intro
We need to encode binary data into signals so the source node can send over a link and the receiving node can decode it back. This is done with [[Network Adaptor]]s.

- [[Return to Zero (RZ)]]
- [[Non-Return to Zero (NRZ)]]
- [[NRZ Inverted (NRZI)]]
