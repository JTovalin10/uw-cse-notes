# Transmission Delay
- Time to put M-bit message "on the wire"
- $$T_{delay} = \frac{M}{R} \text{ seconds}$$
- Depends on message size and link speed
- Example: 1 KB over 1 Mbps = $\frac{8000}{10^6}$ = 8 ms
