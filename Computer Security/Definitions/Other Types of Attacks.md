# Other Types of Attacks

This category encompasses security vulnerabilities beyond standard memory exploits.

## Examples
- **Integer Overflows**: Occur when an arithmetic operation results in a value too large (or small) for the storage type, often leading to bypassed length checks.
- **TOCTOU (Time-of-Check to Time-of-Use)**: A race condition where a system resource is modified between the time it is checked and the time it is used.
- **Timing Attacks**: Side-channel attacks where an attacker extracts secrets (like cryptographic keys) by measuring the time a system takes to perform certain operations.
