# CSE469: Computer Design and Organization

This course explores the design of modern computer systems, focusing on the interface between hardware and software, processor architecture, and the memory hierarchy.

## Topics Covered
- **[[Classes I didnt take/Computer Architecture I/Instruction Set Architecture|Instruction Set Architecture (ISA)]]**: ARMv8 assembly, registers, and instruction encoding.
- **[[CSE469/Computer Arithmetic|Computer Arithmetic]]**: Binary representation, adders, and ALU design.
- **[[Processor Performance|Processor Performance]]**: Throughput, latency, CPI, and Amdahl's Law.
- **[[CSE469/Processor Datapath and Control|Processor Datapath and Control]]**: Single-cycle vs. pipelined designs.
- **[[Pipelining and Hazards|Pipelining and Hazards]]**: Data, structural, and control hazards; forwarding and prediction.
- **[[Memory Hierarchy and Caches|Memory Hierarchy and Caches]]**: Locality, cache organization, and AMAT.
- **[[Advanced Architecture|Advanced Architecture]]**: ILP, Superscalar, VLIW, SIMD/Vector, and GPUs.
- **[[CSE469/Intel Ice Lake Case Study|Intel Ice Lake Case Study]]**: Modern micro-architecture analysis.
