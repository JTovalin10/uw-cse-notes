# CSE469: Computer Design and Organization

Focuses on the high-level design and organization of computer systems. We bridge the gap between digital logic (from [[CSE369 Index|CSE369]]) and software (from [[Hardware & Software Interface/README|CSE351]]).

## Core Components
The five classic components of a computer:
1. **Control**: Orchestrates operations ("who does what when").
2. **Datapath**: Performs operations (Add, subtract, etc.).
3. **Memory**: Stores instructions and data.
4. **Input**: Gathers info from the outside world.
5. **Output**: Provides results.

## Hardware/Software Interface
The **[[Classes I didnt take/Computer Architecture I/Instruction Set Architecture|Instruction Set Architecture (ISA)]]** serves as the abstract interface between the hardware and the lowest-level software (the compiler and OS).
