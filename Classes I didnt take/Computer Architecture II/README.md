# CSE470: Computer Systems Architecture II

Computer Systems Architecture II explores the frontiers of processor design. While CSE469 covers how a basic processor works, CSE470 covers how we make them incredibly fast, efficient, and parallel.

## Core Themes
1. **Parallelism Everywhere**: Extracting ILP (Instruction-Level Parallelism), TLP (Thread-Level Parallelism), and DLP (Data-Level Parallelism).
2. **The Memory Wall**: Managing latency and consistency in complex multi-core hierarchies.
3. **Power as a Constraint**: Designing for energy efficiency and thermal limits.
4. **Specialization**: Moving beyond general-purpose CPUs to GPUs, FPGAs, and ASICs.

## Prerequisites
- Deep understanding of [[Pipelining and Hazards|Pipelining and Hazards]].
- Familiarity with [[CSE351/Memory Fundamentals|Memory Fundamentals]].
