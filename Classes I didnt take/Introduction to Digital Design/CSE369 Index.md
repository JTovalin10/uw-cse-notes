# CSE369 — Introduction to Digital Design

Design and implementation of digital systems, from basic combinational logic and Boolean algebra to sequential circuits, finite state machines, and programmable hardware (FPGAs).

---

## Topics

### Combinational Logic
- [[Combinational Logic]] — logic gates, Boolean algebra, Sum-of-Products, truth tables, and propagation delay

### Verilog Fundamentals
- [[Verilog Fundamentals]] — hardware description language: modules, `assign`, `always_comb`, `always_ff`, blocking vs. non-blocking assignments

### Karnaugh Maps
- [[Karnaugh Maps]] — graphical minimization of Boolean expressions using Gray-code adjacency and grouping rules

### Finite State Machines
- [[Finite State Machines]] — Moore vs. Mealy machines, state diagrams, state encoding, next-state and output logic

### Timing Constraints
- [[Timing Constraints]] — setup time, hold time, clock-to-Q delay, critical path, and metastability

### Building Blocks
- [[Building Blocks]] — multiplexers, decoders, encoders, half/full/ripple-carry adders, and the ALU

### Memory and FPGAs
- [[Memory and FPGAs]] — ROM, SRAM, DRAM, FPGA architecture (LUTs, interconnect, I/O blocks), and the synthesis-to-bitstream workflow
