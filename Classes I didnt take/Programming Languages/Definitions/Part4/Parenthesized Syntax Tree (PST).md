# CSE341: Parenthesized Syntax Tree (PST)

A data structure that represents the nested structure of S-expressions (parenthesized lists) directly. In Trefoil, PSTs serve as an intermediate representation between raw strings and the **[[Classes I didnt take/Programming Languages/Definitions/Part4/Abstract Syntax Tree (AST)|AST]]**, simplifying the parsing process.
