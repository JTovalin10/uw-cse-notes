# CSE341: Closure

A record storing a function together with its defining **[[Environment|Environment]]**. Closures are used to implement **[[Classes I didnt take/Programming Languages/Definitions/Part4/Lexical Scope|Lexical Scope]]**, ensuring that the function has access to the variables that were in scope when it was created.

See: [[Trefoil Functions and Scoping|Trefoil Functions and Scoping]], [[First-Class Functions|First-Class Functions in Trefoil]]
