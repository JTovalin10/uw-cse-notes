# CSE341: Parenthesized Syntax Tree (PST)

Also known as an **S-expression**, it is a representation of code using nested parentheses that maps closely to the program's structure, acting as an intermediate step between tokens and a full **[[Classes I didnt take/Programming Languages/Definitions/Part3/Abstract Syntax Tree (AST)|AST]]**.
