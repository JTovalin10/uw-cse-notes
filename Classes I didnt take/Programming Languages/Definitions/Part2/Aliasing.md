# CSE341: Aliasing

A situation where the same data location can be accessed through multiple names or paths.

### Formal Definition

**Aliasing** occurs when two or more variables or expressions refer to the same memory address in the heap.

### Simplified Explanation

Two different names for the exact same physical thing. Changes made through one name are visible through the other.
