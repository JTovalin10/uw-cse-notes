# CSE341: Type Constructor

A feature of a type system that creates a new type from existing types.

### Formal Definition

An n-ary operator that takes types as arguments and returns a type. Examples include `*` for tuples, `list`, and `option`.

### Simplified Explanation

A "factory" for types. For example, `list` isn't a type by itself, but it can make `int list` or `string list`.
