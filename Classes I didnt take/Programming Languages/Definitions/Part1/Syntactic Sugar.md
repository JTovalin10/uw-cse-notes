# CSE341: Syntactic Sugar

Syntax within a programming language that is designed to make things easier to read or to express, but which can be defined in terms of more fundamental language constructs.

### Formal Definition

A feature that simplifies the syntax without adding new expressive power to the language semantics.

### Simplified Explanation

A "shortcut" way of writing code that the compiler translates into a more basic form.
