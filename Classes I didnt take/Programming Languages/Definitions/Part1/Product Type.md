# CSE341: Product Type

A type formed by combining other types such that a value of the product type contains a value from each of the component types.

### Formal Definition

The Cartesian product of types $T_1 \times T_2 \times \dots \times T_n$. In OCaml, this is represented by **[[Tuple|Tuples]]** and **[[Record|Records]]**.

### Simplified Explanation

An "Each-Of" type where you have one of everything listed.

See: [[Lists and Tuples|Lists and Tuples]]
