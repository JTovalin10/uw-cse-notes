# CSE341: Aliasing

A situation where multiple names or references point to the same location in memory or the same data value.

### Formal Definition

The existence of two or more distinct identifiers that refer to the same mutable or immutable object.

### Simplified Explanation

When one piece of data has two different "nicknames."
