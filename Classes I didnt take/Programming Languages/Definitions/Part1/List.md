# CSE341: List

A compound data type that holds any number of values (pieces), all of which must have the same type.

### Formal Definition

A list of type $t$ is denoted as $t \text{ list}$. It is either empty ($[]$) or a head element of type $t$ followed by a tail which is itself a $t \text{ list}$.

### Simplified Explanation

An ordered collection of items where everything in the list must be the same kind of thing.
