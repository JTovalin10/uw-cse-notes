# CSE341: Closure

A record storing a function together with its defining **[[CSE341/Definitions/Part4/Environment|Environment]]**. Closures are used to implement **[[CSE341/Definitions/Part4/Lexical Scope|Lexical Scope]]**, ensuring that the function has access to the variables that were in scope when it was created.
