# CSE341: Interpreter

A program that directly executes instructions written in a programming or scripting language, without requiring them previously to have been compiled into a machine language program. In Trefoil, the interpreter is a recursive function that evaluates an **[[CSE341/Definitions/Part4/Abstract Syntax Tree (AST)|AST]]** within a given **[[CSE341/Definitions/Part4/Environment|Environment]]**.
