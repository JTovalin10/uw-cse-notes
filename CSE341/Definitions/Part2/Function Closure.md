# CSE341: Function Closure

A value containing both a function's code and the environment in which it was defined.

### Formal Definition

A **Function Closure** is a pair $(C, E)$ where $C$ is the function code and $E$ is the lexical environment mapping free variables to values at the time of definition.

### Simplified Explanation

A function "carrying its backpack" of variables it needs from where it was born.
