# CSE341: Partial Application

The process of applying a curried function to some, but not all, of its arguments.

### Formal Definition

**Partial Application** occurs when a curried function is called with fewer arguments than it expects, resulting in a closure that represents the remaining computation.

### Simplified Explanation

Giving a function some of its "fuel" now and getting back a specialized function that waits for the rest.
