# Course: First Class Function

A property of a programming language where functions are treated as values, meaning they can be passed as arguments, returned as results, and stored in data structures.

### Formal Definition
A **[[CSE341/Definitions/Part2/First Class Function|First Class Function]]** is an entity that supports all the operations generally available to other entities, such as being passed as an argument, returned from a function, and assigned to a variable.

### Simplified Explanation
Functions can be used just like any other value (like integers or strings).
