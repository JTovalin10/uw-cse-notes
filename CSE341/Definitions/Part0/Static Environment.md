# CSE341: Static Environment

### Formal Definition

A mapping from variable names to their corresponding types, maintained during the compile-time type-checking phase.

### Simplified Explanation

A dictionary the compiler uses to remember the data types of variables before the program runs, ensuring you don't accidentally add an integer to a string or call a method that doesn't exist.
