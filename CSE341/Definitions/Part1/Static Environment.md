**[[CSE341/Definitions/Part1/Static Environment|Static Environment]]**: A mapping from variable names to their types, used during the type-checking phase of compilation.
### Formal Definition
A compile-time context $\Gamma$ that tracks the types of all currently bound identifiers.
### Simplified Explanation
The "dictionary" the compiler uses to make sure you're using variables correctly (e.g., not adding a string to an integer).
