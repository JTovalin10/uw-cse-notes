# CSE341: Record

A compound data type that holds a fixed number of values, each associated with a named field.

### Formal Definition

An "each-of" type `{f1 : t1; ...; fn : tn}` where values are accessed via field labels $f_i$ rather than position.

### Simplified Explanation

A collection of named pieces of data, similar to a row in a spreadsheet or a simple object without methods.
