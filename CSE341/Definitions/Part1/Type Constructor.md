**[[CSE341/Definitions/Part1/Type Constructor|Type Constructor]]**: A feature of a type system that creates a new type from existing types.
### Formal Definition
A n-ary operator that takes types as arguments and returns a type. Examples include `*` for tuples, `list`, and `option`.
### Simplified Explanation
A "factory" for types. For example, `list` isn't a type by itself, but it can make `int list` or `string list`.
