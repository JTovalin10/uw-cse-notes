**[[CSE341/Definitions/Part1/Product Type|Product Type]]**: A type formed by combining other types such that a value of the product type contains a value from *each* of the component types.
### Formal Definition
The Cartesian product of types $T_1 \times T_2 \times \dots \times T_n$. In OCaml, this is represented by **[[CSE341/Definitions/Part1/Tuple|Tuples]]** and **[[CSE341/Definitions/Part1/Record|Records]]**.
### Simplified Explanation
An "Each-Of" type where you have one of everything listed.
