# CSE341: Structural Equality

A form of equality that checks if two values have the same content and shape, regardless of whether they are the same object in memory.

### Formal Definition

Represented by the `=` operator in OCaml. It recursively checks if all components of a data structure are equal.

### Simplified Explanation

Two things are "equal" if they look exactly the same inside.
