# CSE 312: Probability Mass Function (PMF)

#Definition The **PMF** (Probability Mass Function) gives the probability that a discrete random variable takes on a specific value. Formally:

$$p_X(k) = P(X = k)$$

$$P(X = a)$$

## Properties of PMF
- Any value within $p_X(k)$ must be greater than or equal to 0
- The sum of all possible probabilities must equal 1
