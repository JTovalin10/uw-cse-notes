# CSE 312: Distribution (Counting)

#Definition A **distribution** in the counting sense describes how objects are assigned to recipients. If we have $n$ distinguishable objects to distribute among $k$ recipients, then the number of ways to do so is:

$$k^n$$

## Industry Standard Terms

- **Distribution (counting sense)** → "distributing distinguishable objects into distinguishable boxes." This is the same $k^n$ count as the number of functions from an $n$-set to a $k$-set.
