# Bernoulli Distribution

An indicator random variable $X$ representing a single trial with probability $p$ of success. $X=1$ if success, $X=0$ if failure.

**Notation:** $X \sim \text{Ber}(p)$

**Probability Mass Function (PMF):**
$$p_X(1) = p$$
$$p_X(0) = 1 - p$$

**Expectation:**
$$\mathbb{E}[X] = p$$

**Variance:**
$$\text{Var}(X) = p(1 - p)$$
