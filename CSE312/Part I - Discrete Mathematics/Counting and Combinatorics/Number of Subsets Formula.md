# CSE 312: Combinations (Choose Formula)

**$C(n,k)$**, also written $\binom{n}{k}$, counts the number of ways to choose $k$ items from $n$ items where order does not matter.

$$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

The name "Number of Subsets Formula" reflects that $\binom{n}{k}$ also counts the number of distinct size-$k$ subsets of an $n$-element set, since a subset is an unordered selection.
