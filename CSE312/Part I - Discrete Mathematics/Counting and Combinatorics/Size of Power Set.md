# CSE 312: Size of Power Set

The **power set** $\mathcal{P}(S)$ is the set of all subsets of $S$, and its size is:

$$|\mathcal{P}(S)| = 2^{|S|}$$

This is because each element of $S$ is either included or excluded — 2 choices per element — and by the product rule, multiplying 2 across all $|S|$ elements gives $2^{|S|}$.
