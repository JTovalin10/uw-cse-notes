The set $\sum^*$ of strings over the alphabet $\sum$
- Example: $\set{0,1}^*$ 
	- 0, 1, 01, 10, 11, 000, 001, ...
[[Strings]]

$$\sum^*$$
$$\text{Is defined as}$$
$$\text{Basis: } \epsilon \in \sum^* \quad \text{Epsilon is the empty string ""}$$
$$\text{Recursion: } w \in \sum^*, a \in \sum , \text{then } wa \in \sum^*$$
