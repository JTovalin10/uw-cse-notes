**Length**:
len(nil) := 0
len(a :: L) := len(L) + 1.   for any L $\in$ List and a $\in \mathbb{Z}$


**Concatenation**:
concat(nil, R) := R.  for any R $\in$ List
or concat(R, nil) := R
concat(a :: L, R) := a :: concat(L, R). for any R $\in$ list and a $\in \mathbb{z}$
