# CSE391: System & Software Tools

This repository contains structured notes for the University of Washington's **CSE391** course, also known as "System & Software Tools".

## Course Overview

CSE391 is often called the "missing semester" of a CS major. It covers a collection of tools and topics not typically addressed in other classes but essential for every computer scientist and software engineer.

### Key Topics
- **[[Linux Fundamentals/Introduction to Linux|Linux and its Command Line Interface (CLI)]]**
- **Shell Commands**
- **[[Streams Redirection and Pipes/Input Output Redirection|Input/Output Redirection and Pipes]]**
- **[[Git/Git Fundamentals|Version Control and Collaboration using Git]]**
- **[[Regular Expressions/Regex Basics|Regular Expressions, grep, and sed]]**
- **[[Users Groups and Permissions/Unix Permissions|Unix Users, Groups, and Permissions]]**
- **[[Bash Scripting/Bash Scripting Basics|Basic Shell Scripting]]**

For a detailed breakdown of all topics, see the **[[CSE391 Index]]**.
