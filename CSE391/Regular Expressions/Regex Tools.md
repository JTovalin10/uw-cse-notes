# CSE391: Regex Tools and Applications

Regular expressions are versatile tools used across many areas of software engineering.

## Common Applications
- **Input Validation:** Ensuring user input (emails, phone numbers, credit cards) follows a specific format.
- **Data Scraping:** Extracting specific information from logs, web pages, or large datasets.
- **Find and Replace:** Mass editing code or text files using tools like `sed` or IDE search features.
- **Search:** Quickly finding lines or files containing specific patterns.

## Recommended Tools
- **regex101.com**: An excellent interactive tool for writing, testing, and debugging regular expressions.
- **regexr.com**: Another powerful regex tester with community patterns.
- **Regex Golf**: A challenging game to improve your regex skills.

## Regex Flavors
Be aware that there are different "flavors" of regex (e.g., PCRE, JavaScript, Java, GNU BRE/ERE). While the core syntax is similar, some special characters or features may vary. In this class, we focus on **GNU Extended Regular Expressions (ERE)** used by `grep -E`.
