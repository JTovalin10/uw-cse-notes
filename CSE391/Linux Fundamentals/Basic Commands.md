# CSE391: Common Command Definitions

Short definitions for basic commands used in CSE391.

### [[cat]]
Concatenates and displays file content.

### [[echo]]
Displays text or variables.

### [[head]]
Outputs the first part of a file.

### [[tail]]
Outputs the last part of a file.

### [[wc]]
Counts lines, words, and characters.

### [[sort]]
Sorts lines of text.

### [[uniq]]
Filters out duplicate lines.

### [[find]]
Searches for files in a directory.

### [[cut]]
Extracts sections from lines of text.
