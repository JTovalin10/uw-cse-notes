# Operating System

An **Operating System (OS)** is the fundamental software that manages a computer's hardware and software resources. It acts as an intermediary between the computer hardware and the applications that users run.

## Key Responsibilities of an OS
- **Hardware Management:** Controls the CPU, memory (RAM), disk drives, and peripheral devices (keyboard, monitor, network cards).
- **Process Management:** Allocates CPU time to various running programs and ensures they don't interfere with each other.
- **Memory Management:** Keeps track of each byte of memory and assigns it to specific processes as needed.
- **File System:** Organizes and stores data on persistent storage (e.g., SSDs, Hard Drives).
- **User Interface:** Provides a way for users to interact with the machine (GUI or CLI).
- **Security & Permissions:** Manages user accounts and controls who can access specific files or run specific programs.

---

## The Unix and Linux Family
Unix-like systems (including Linux and macOS) share a common philosophy and structure.

| System | Description |
| :--- | :--- |
| **Unix** | The original proprietary OS developed at Bell Labs in the 1970s. |
| **Linux** | An open-source, Unix-like kernel created by Linus Torvalds in 1991. Used by Ubuntu, Fedora, Debian, etc. |
| **macOS** | A proprietary Unix-based system from Apple (based on Darwin/BSD). |
| **Windows** | A separate OS family, though it now supports Linux via **WSL (Windows Subsystem for Linux)**. |

---

## The Kernel
The **Kernel** is the heart of the operating system. It is the program that has complete control over everything in the system. It is usually the first program loaded on startup and manages all low-level hardware interactions.
