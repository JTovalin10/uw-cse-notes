# CSE391: Operating System

An **[[Operating System]]** is the core software that manages a computer's hardware and software resources (memory, processors, peripherals, users).

## Examples
- **Linux**: Open-source, widely used on servers and supercomputers.
- **macOS**: Unix-based (BSD lineage).
- **Windows**: Integrated with Linux via **[[Windows Subsystem for Linux|WSL]]**.
- **Android**: Based on the Linux kernel.

## Key Functions
- Managing files and processes.
- Handling user permissions and security.
- Providing a **[[Shell]]** or GUI for user interaction.
