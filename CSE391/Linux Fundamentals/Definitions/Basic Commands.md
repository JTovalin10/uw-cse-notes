# CSE391: Common Command Definitions

Short definitions for basic commands used in CSE391.

### cat
**[[cat]]**: Short for "concatenate," used to read and output the entire content of one or more files to the terminal.

### echo
**[[echo]]**: Displays a line of text or the value of a variable to the standard output.

### head
**[[head]]**: Displays the first few lines (default 10) of a file.

### tail
**[[tail]]**: Displays the last few lines (default 10) of a file.

### wc
**[[wc]]**: Short for "word count," it counts lines, words, and characters in a file.

### sort
**[[sort]]**: Reorders the lines of a file alphabetically or numerically.

### uniq
**[[uniq]]**: Filters out adjacent duplicate lines. Often used after `sort`.

### find
**[[find]]**: A powerful tool for searching the filesystem for files that match specific criteria (name, size, type).

### cut
**[[cut]]**: Extracts specific columns or fields from a text file, usually delimited by a character like a comma or tab.
