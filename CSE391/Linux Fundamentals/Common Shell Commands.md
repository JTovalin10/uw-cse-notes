# CSE391: Common Shell Commands

These are essential utility commands used throughout the course for file manipulation and data processing.

| Command | Description |
|---------|-------------|
| **[[cat]]** | Concatenate and print files to stdout. |
| **[[echo]]** | Print text or variables to the terminal. |
| **[[grep]]** | Search text using regular expressions. |
| **[[sed]]** | Stream editor for filtering and transforming text. |
| **[[sort]]** | Sort lines of text files. |
| **[[uniq]]** | Report or omit repeated lines (requires sorted input). |
| **[[wc]]** | Print newline, word, and byte counts for each file. |
| **[[head]]** | Output the first part of files. |
| **[[tail]]** | Output the last part of files. |
| **[[find]]** | Search for files in a directory hierarchy. |
| **[[cut]]** | Remove sections from each line of files. |
