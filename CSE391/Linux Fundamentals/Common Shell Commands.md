# CSE391: Common Shell Commands

These are essential utility commands used throughout the course for file manipulation and data processing.

| Command | Description |
|---------|-------------|
| **[[CSE391/Linux Fundamentals/Commands/cat\|cat]]** | Concatenate and print files to stdout. |
| **[[CSE391/Linux Fundamentals/Commands/echo\|echo]]** | Print text or variables to the terminal. |
| **[[CSE391/Linux Fundamentals/Commands/grep\|grep]]** | Search text using regular expressions. |
| **[[CSE391/Sed/Sed Commands\|sed]]** | Stream editor for filtering and transforming text. |
| **[[CSE391/Linux Fundamentals/Commands/sort\|sort]]** | Sort lines of text files. |
| **[[CSE391/Linux Fundamentals/Commands/uniq\|uniq]]** | Report or omit repeated lines (requires sorted input). |
| **[[CSE391/Linux Fundamentals/Commands/wc\|wc]]** | Print newline, word, and byte counts for each file. |
| **[[CSE391/Linux Fundamentals/Commands/head\|head]]** | Output the first part of files. |
| **[[CSE391/Linux Fundamentals/Commands/tail\|tail]]** | Output the last part of files. |
| **[[CSE391/Intermediate Command Line/Find and Cut#find\|find]]** | Search for files in a directory hierarchy. |
| **[[CSE391/Intermediate Command Line/Find and Cut#cut\|cut]]** | Remove sections from each line of files. |
