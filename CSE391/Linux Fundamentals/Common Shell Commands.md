# CSE391: Common Shell Commands

These are essential utility commands used throughout the course for file manipulation and data processing.

| Command | Description |
|---------|-------------|
| **[[Linux Fundamentals/Commands/cat\|cat]]** | Concatenate and print files to stdout. |
| **[[Linux Fundamentals/Commands/echo\|echo]]** | Print text or variables to the terminal. |
| **[[Linux Fundamentals/Commands/grep\|grep]]** | Search text using regular expressions. |
| **[[Sed/Sed Commands\|sed]]** | Stream editor for filtering and transforming text. |
| **[[Linux Fundamentals/Commands/sort\|sort]]** | Sort lines of text files. |
| **[[Linux Fundamentals/Commands/uniq\|uniq]]** | Report or omit repeated lines (requires sorted input). |
| **[[Linux Fundamentals/Commands/wc\|wc]]** | Print newline, word, and byte counts for each file. |
| **[[Linux Fundamentals/Commands/head\|head]]** | Output the first part of files. |
| **[[Linux Fundamentals/Commands/tail\|tail]]** | Output the last part of files. |
| **[[Intermediate Command Line/Find and Cut#find\|find]]** | Search for files in a directory hierarchy. |
| **[[Intermediate Command Line/Find and Cut#cut\|cut]]** | Remove sections from each line of files. |
